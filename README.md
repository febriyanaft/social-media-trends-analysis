# Analisis Tren Viral Social Media

## Deskripsi
Analisis tren dan engagement konten viral di berbagai platform media sosial 
menggunakan Google Sheets berdasarkan dataset dari Kaggle.

## Dataset
Sumber: Kaggle - Viral Social Media Trends & Engagement Analysis
Jumlah data: 5.000 baris

## Tools
- Google Sheets (SUMIF, visualisasi chart)

## Insight
1. YouTube memiliki total views tertinggi sebagai platform video utama
2. #Fitness menjadi hashtag paling banyak ditonton, mencerminkan 
   pergeseran minat audiens ke konten yang bermanfaat
3. USA menjadi region dengan engagement tertinggi sebagai pasar digital terbesar
